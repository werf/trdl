## Get the task log


| Method | Path |
|--------|------|
| `GET` | `/task/:uuid/log` |

### Parameters

* `limit` (url pattern, optional) — Limit of characters.
* `offset` (url pattern, optional) — Offset.
* `uuid` (url pattern, required) — Task UUID.

### Responses

* 200 — OK.

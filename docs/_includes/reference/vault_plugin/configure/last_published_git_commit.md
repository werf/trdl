Read or delete the last published Git commit.

## Get the last published Git commit


| Method | Path |
|--------|------|
| `GET` | `/configure/last_published_git_commit` |


### Responses

* 200 — OK. 


## Delete the last published Git commit


| Method | Path |
|--------|------|
| `DELETE` | `/configure/last_published_git_commit` |


### Responses

* 204 — empty body.
